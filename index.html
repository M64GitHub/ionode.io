Instant Hardware Access

GPIO, ADC, PWM, UART, sensors, actuators. Request/reply over NATS. 66ms round trip.

Built to be Hacked

Add any sensor in minutes. Zero NATS code. Your hardware, your rules.

The HAL Protocol

Two lines of NATS. Read anything. Control anything.

nats cli 
# Read a sensor
nats req ionode-01.hal.temperature ""      # → 24.5

# Toggle a relay
nats req ionode-01.hal.fan.set "1"         # → ok

# Raw GPIO
nats req ionode-01.hal.gpio.4.get ""       # → 0

# Discover the fleet
nats req _ion.discover ""                  # → all nodes respond
